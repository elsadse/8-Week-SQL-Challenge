# 8 Week SQL Challenge – Mes Solutions

Ce dépôt contient mes solutions aux études de cas du défi SQL de huit semaines proposé sur https://8weeksqlchallenge.com/ créer par [Danny Ma](https://www.linkedin.com/in/datawithdanny/).


Ce dépôt montre ma capacité à :
- Analyser des problématiques business réelles avec SQL
- Écrire des requêtes propres, performantes et lisibles
- Utiliser des techniques avancées (window functions, CTE, pivoting, regex, joins complexes, etc...)
- Présenter clairement mes raisonnements et résultats

## 🛠️ Technologies & Outils utilisés

- **SGBD principal** : PostgreSQL 
- **Environnement** : pgAdmin, VsCode
- **Techniques SQL mises en avant** :
  - CTE (Common Table Expressions) & CTE récursives
  - Window functions (``ROW_NUMBER``, ``RANK``, ``LAG/LEAD``, etc...)
  - Agrégations conditionnelles (``SUM(CASE WHEN …)``, ``AVG(…)``, ``COUNT(…)``, etc...)
  - Pivoting / unpivoting
  - Traitement de dates & intervals
  - Jointures complexes & self-joins
  - Regex & pattern matching
  - Gestion des données manquantes / nettoyage
